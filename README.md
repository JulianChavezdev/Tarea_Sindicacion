# RSS News Aggregator Project

A streamlined web-based news aggregator designed to provide a clean
user interface and a functional XML news channel for seamless content subscription.

### 1. Web interface
The project features a simple and clean landing page (`index.html`) that serves as the primary front end:

* **Main Section:** Contains a brief explanation of the chosen theme and the aggregator's purpose.

### How to use the Aggregator
To follow our news feed and stay updated, follow these steps:

1. **Copy the URL** of our XML file directly from the live site.
2. **Open your preferred content aggregator** (such as [Feedly]
 (https://feedly.com) or Old Reader).
 3. Click on **"Add Content"** or **"Follow"** and paste the copied URL.
 4. The news items will **automatically appear** and update in your personal feed.
